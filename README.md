

## Como executar:

### Primeiro baixe o projeto e suas dependencias

1. No seu terminal, vai para a pasta backend, execute os comandos.
    + cd AuthApp (para entrar no diretorio de execução DOTNET)
    + dotnet run ( para rodar em http://localhost:7000 o server interno)
3. Apos iniciar o server , vai para a pasta frontend, execute os comandos.
    + ng serve (para rodar o front end)
    + Em seu navegador vá para http://localhost:4200/ (onde estara executando a aplicação)
